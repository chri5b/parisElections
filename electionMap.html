Paris 2012 Election Results

Where did each candidate in the 2012 French presidential elections perform best in Paris?

Using the detailed voting station data published as part of the Open Data initiative by the French Government, this map plots a circle for each voting station in Paris. The colour indicates which of the two candidates selected had the most votes. Size of the circle indicates the total number of voters for both candidates and is scaled to the maximum number of votes for both candidates recieved.

Implementation uses map data from cloudmade, and uses leaflet.js for the map UI. Node does the datamangling. Geocoded information for each of the Paris voting stations can be found here
